Step 3: Install Thistle Tools

TRH on workstation, TUC on device

Install TRH (Your Computer)

# Thistle Release Helper - packages and signs updates curl -LO "https://downloads.thistle.tech/embedded-client/1.5.0/\ trh-1.5.0-[YOUR-ARCH].gz" gunzip trh-*.gz && chmod +x trh-* && sudo mv trh-* /usr/local/bin/trh

Install TUC (BeagleY-AI via SSH)

# Thistle Update Client - receives and applies updates curl -LO "https://downloads.thistle.tech/embedded-client/1.5.0/\ tuc-1.5.0-aarch64-unknown-linux-musl.gz" gunzip tuc-*.gz && chmod +x tuc-* && sudo mv tuc-* /usr/local/bin/tuc

Step 4: Initialize Project

Create signing keys and configuration

# On your workstation - set API token export THISTLE_TOKEN="tht_proj_xxxxxxxxxxxxxx" # Initialize project with signing keys mkdir ~/beagley-demo && cd ~/beagley-demo trh init --persist="/boot" # This creates: config.json # Device enrollment config manifest.json # Update template release_keys/ # Ed25519 keypair (KEEP SAFE!)
# Deploy config to BeagleY-AI scp config.json debian@beagley-ota.local:/tmp/ ssh debian@beagley-ota.local sudo mv /tmp/config.json /boot/tuc-config.json

Step 5: Create Update Script

Simple LED blink to prove OTA works

Optional: Connect LED to Pin 8 (GPIO14) → LED → Resistor → Pin 6 (GND)
mkdir -p ~/beagley-demo/files cat > ~/beagley-demo/files/blink.sh << 'EOF' #!/bin/bash echo "🎉 OTA Update Success! Version 1.0" echo "Press Ctrl+C to stop" # Blink LED on GPIO14 while true; do gpioset 0 14=1 # ON sleep 0.5 gpioset 0 14=0 # OFF sleep 0.5 done EOF chmod +x ~/beagley-demo/files/blink.sh

Step 6: Sign & Release Update

Package, sign, and upload to cloud

# Package and sign with your private key cd ~/beagley-demo trh prepare --target=./files --file-base-path=/opt/demo # Upload to Thistle cloud trh release Uploading manifest... Release successful! Version: 1
